# 介绍下webpack热更新原理，是如何做到在不刷新浏览器的前提下更新页面?

`模块热替换(hot module replacement 或 HMR)`是 webpack 提供的最有用的功能之一。它允许在运行时更新所有类型的模块，而无需完全刷新。
